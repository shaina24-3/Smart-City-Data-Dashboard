# Smart-City-Data-Dashboard
A Smart City Data Dashboard that fetches real-time and publicly available data from global smart cities using open APIs. The dashboard visualizes key urban metrics like air quality, weather, traffic, energy consumption, and waste management to support better decision-making and promote data-driven urban planning.
